INSTRUCTIONS

  1. The aim of the game is to score more points than the opponent.
  2. The active player will roll the die and a random number from 1 to 6 will appear on the die, which will be your current score.
  3. The current score will add up, every time you roll the die.
  4. On clicking the 'Hold' button, your summed up current score will be added to your total score, and your turn will end.
  5. But if a player rolls a '1' on the die, the player turn ends, loosing all the points in the current score. Hence, '1' means Danger!.
  6. The player with the maximum points at the end of the game will be the 'Winner'.

Time Trial

  1. In this game mode, apart of each other, you also have to race against time.
  2. The Time Limit can be set in the begining of the game.
  3. The game will end once the time is over!

Beat The Score

  1. In this game mode, players are required to beat the score limit set in the beginning, before the opponent to win.

Rounds

  1. In this game mode, the number of turns/rounds are set in the beginning.
  2. Players need to get more points than the opponent in the set number of rounds to win
  3. The players can only roll a maximum of 3 times in a single round.
  4. In this game mode rolling a '1' does not result in loosing the summed up current score. '1' is a safe number.
  5. But trying to roll more than 3 times will result in an automatic change of turns and loosing the current score.
